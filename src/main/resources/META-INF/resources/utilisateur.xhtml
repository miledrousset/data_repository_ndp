Les régles à respecter
  • Format du date : dd/M/yyyy. Exemple : 22/11/2021
  • Les mots clés doivent être séparer par ' ### '. Exemple : MOT_CLE_1 ### MOT_CLE_2 ###
  • Chaque mot clé est sous forme : label#URI. Avec URI est un champs optionnel.
  • Touts les carectères sont autorisés.
  • Le nombre des clonnes doivent être respecter. Si un champs vide est doit etre syboliser par ''.
